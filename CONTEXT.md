# CONTEXT — хендофф

Обновлено: 2026-07-16

## Что это
Мобильный прототип главного экрана Ozon Банка для бизнеса (веб-прототип живёт отдельно в `~/Прототип`, репо `sukhovdes/bank2.0`). Один экран, фуллскрин-мобайл (смотрится с телефона), интерактив — только скролл.

Макеты: Figma `ZRNYwdJic4yxtpdy2QXNQh`, страница «мобайл» — фреймы `Start` (82185:2615) и `Scroll` (82185:2921) = один экран в двух состояниях скролла.

## Запуск / деплой
- `npm run dev` → http://localhost:5174 (host: true — можно открыть с телефона по IP).
- Репо: `git@github.com:sukhovdes/bankapp.git`. Деплой: Vercel, https://bankapp-liart.vercel.app (SPA-fallback в `vercel.json`).

## Светлая версия — /light и /new_button
- `src/Light.jsx`, роутинг без библиотек: `main.jsx` смотрит на `location.pathname`.
- `/new_button` = `<Light variant="buttons" />`: вместо карточки «Быстрые действия» — ряд матовых кнопок под балансом (Пополнить/Платеж/Между счетам/Все), лента: Ожидают действия → баннеры → остальное.
- Редизайн элементов (фрейм 82227:3016): «Ожидают действия» — плитки на `--bg-secondary` с серым hairline-делителем; «Новый продукт» — светлая плитка ТОЛЬКО на /light (на тёмной главной остался glass — серый вариант нечитаем на синем градиенте, стили разведены через `.light`-скоуп).
- По фрейму «Мой новый» (82216:1875): шапка с фото-аватаром (`public/products/avatar.png`), тёмный баланс, бейдж с обводкой; лента — те же блоки, что на тёмной главной (компоненты экспортированы из App.jsx и переиспользуются).
- Металлический фон: CSS-эллипсы по `bg / silver` (82216:1810), блики дрейфуют (`@keyframes metal-drift-*`), фейд снизу в #F5F7FA.
- Уведомления над балансом (82222:2149): скрыты при загрузке, выезжают через 900мс (grid-rows 0fr→1fr), крестик схлопывает строку (удаление по setTimeout 380мс — transitionend по grid-rows ненадёжен), после закрытия обеих блок сворачивается.
- Гоча: у SVG-экспорта из Figma бывают артефакты инстансов — запечённые fill-opacity (ключ был 0.4 — вычищено) и подменённые слои (иконка СБП выгрузилась как ai_stars — забракована). Проверять содержимое SVG после скачивания.

## Структура
- `src/App.jsx` — все секции экрана одним файлом: TopBar (фикс), Balance, Widgets (карусель), SystemAlert (пульс красного свечения ×3), QuickActions, PendingActions, IncomeIsland (прогноз дохода), ProfitIsland, PayoutTile, MonthCalendar, TabBar (фикс). «Заявка на кредит» удалена по фрейму «Мой новый» (82206:2887).
- `src/styles.css` — токены + все стили (BEM-иш).
- `src/components/Icon.jsx` — иконки через CSS-mask: `url(/icons/<name>.svg)` + backgroundColor. Все SVG скачаны из Figma.
- `public/bg-header-gradient.svg` — фоновый градиент шапки: эллипсы из Figma, но backdrop-blur заменён на общий `feGaussianBlur stdDeviation=60` (без шва, работает в Safari). Инлайнится в DOM через `?raw`-импорт в App.jsx.
- `public/products/` — логотипы (wildberries, ozon, `logo_unknown.svg` — второй маркетплейс, имя не опознано).

## Гочи
- Скролл — НАТИВНЫЙ (body): не заворачивать приложение в `position:fixed`-контейнер с внутренним скроллом, иначе Safari на iOS не сворачивает панели и режет высоту. Шапка/таббар/фон — `position:fixed`, слушатель скролла — на `window`.
- Из фрейма «Мой новый» пока НЕ перенесены: «Последние операции», обновлённое «Ожидают действия» (счётчик 2, аватар).
- Тени карточек (`--card-shadow`, свечение rgba(77,103,198,.3)) **временно отключены** по просьбе пользователя — см. `.card--shadow` в styles.css.
- Подложка под фикс-шапкой (`.topbar::before`) появляется только при скролле (>12px, класс `topbar--scrolled` из onScroll в App.jsx). Текущая интенсивность 0.85/0.55 — согласована.
- В превью-браузере программный `scrollTop` НЕ вызывает scroll-событие — проверять скролл-поведение руками или через `dispatchEvent`.
- Иконка слева в бейдже «Общий баланс» (status_points, #232323) в макете невидима на тёмном — намеренно не вставлена. Чеврон справа от суммы баланса тоже опущен (в макете не виден).
- Календарь: Июль 2026, сетка 27–31 (пред. месяц, серые) + 1–31 + 1–6 (след., серые), у всех подпись «4 000 ₽», розовые точки на 12, 13, 18, 21, 26.
- Safe-area: отступы через `env(safe-area-inset-*)`, на десктопе = 0, на телефоне всё съедет ниже — это норм.
