# Прототип Ozon Банк для бизнеса — мобильное приложение

React + Vite. Запуск: `npm run dev` (порт 5174). Дизайн в Figma `ZRNYwdJic4yxtpdy2QXNQh`, страница «мобайл» (node 82185-606).

## ⚡️ В начале каждой сессии
**Сразу прочитай `CONTEXT.md`** в корне проекта — это актуальный хендофф-файл (состояние, структура, гочи). Делай это без напоминаний пользователя.

## ⛔️ Правило №1 (нарушать нельзя): иконки только из Figma

- ВСЕ иконки берём из Figma-макета (`download_assets` / `get_design_context`), складываем в `public/icons/`.
- НЕ рисовать инлайновые/самодельные SVG-иконки.
- Если нужной иконки нет в макете или её не получается достать — **остановиться и спросить пользователя**, не подставлять свою.

## Прочее
- Дизайн-токены — CSS-переменные в `src/styles.css` (шрифт Onest, синий `#005BFF`, текст `#070707`, фон `#F2F5F9`).
- Иконки перекрашиваются через CSS-mask (компонент `src/components/Icon.jsx`), сами SVG — из Figma.
- Логотипы и ассеты — в `public/products` и `public/icons`.
- Коммитим реже — пачками, по слову пользователя «сохрани».
